# Sales Dashboard (Next.js + Tailwind + Recharts)

Deployed on Vercel from GitHub with zero local setup.

## Data
Place your tidy dataset at `public/data/sales_tidy.json`.
The dashboard fetches from `/data/sales_tidy.json`.

## Dev scripts
- `npm run build`
- `npm run start`

(You don't need to run locally; Vercel will run these.)
